# AI Resume Analyzer

## Goal

An AI-powered web application that analyzes resumes against job descriptions, identifies missing skills, calculates a match score, and provides improvement suggestions.

## Tech Stack (Planned)

- React
- FastAPI
- SQLite (later PostgreSQL)
- Python
- Tailwind CSS (later)
